# BigWaves Performance Dashboard

Multi-tenant dashboard voor BigWaves AI-bureau. Klanten kunnen inloggen met unieke credentials en hun eigen performance metrics bekijken.

## Deploy op Streamlit Cloud

1. Fork deze repo
2. Ga naar [share.streamlit.io](https://share.streamlit.io)
3. Koppel de repo
4. Deploy!

## Lokaal draaien

```bash
pip install -r requirements.txt
streamlit run dashboard.py
```
